Instalacion y uso minimo

La demo usa el build publicado. El mismo contrato sirve para Netlify, examples y consumidores del paquete.

Quick start 
import { defineAllComponents, setTheme } from 'no-dep-ds';
import 'no-dep-ds/styles.css';

defineAllComponents();
setTheme('light');

document.body.innerHTML = `
  <nds-card elevated>
    <nds-stack gap="4">
      <nds-heading level="3">Hello</nds-heading>
      <nds-text>Build once, publish anywhere.</nds-text>
      <nds-button>Primary action</nds-button>
    </nds-stack>
  </nds-card>
`;

Que valida esta demo

El objetivo es que el despliegue sea util para quien consume el DS, no solo para una smoke test interna.

  • Todas las paginas cargan componentes reales desde /dist.
  • El selector de tema persiste entre paginas con localStorage.
  • Los snippets muestran markup tal como se escribiria en HTML.
  • Las demos interactivas registran eventos publicos del host.
  • La seccion DOM modes separa el registro por pagina para evitar redefiniciones.
