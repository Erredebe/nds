Instalacion y uso minimo

El contrato de arranque es pequeno: importas el paquete, registras componentes y aplicas tema.

Quick start 
import { defineAllComponents, setTheme } from 'no-dep-ds';
import 'no-dep-ds/styles.css';

defineAllComponents();
setTheme('light');

document.body.innerHTML = `
  <nds-card elevated>
    <nds-stack gap="4">
      <nds-heading level="3">Portable UI</nds-heading>
      <nds-text>Mismo paquete para marketing, producto y apps internas.</nds-text>
      <nds-button>Primary action</nds-button>
    </nds-stack>
  </nds-card>
`;
