nds-select Events: nds-input / nds-change

Select

Control de seleccion con opciones serializadas, placeholder y soporte para formularios asociados.

Opciones y placeholder

La prop options deja authoring rapido en HTML y mantiene el componente listo para formularios reales.

Markup 
<nds-select
  label="Stack prioritario"
  placeholder="Selecciona un stack"
  value="astro"
  options="html:HTML|react:React|vue:Vue|astro:Astro"
></nds-select>

Eventos

Usa eventos del host para reaccionar al valor seleccionado sin inspeccionar el <select> interno.

Cambia el primer select para ver el valor emitido.
