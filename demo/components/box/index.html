nds-box surface / padding / radius

Box

Contenedor ligero para aplicar fondo, borde, padding y radio desde tokens semanticos.

Superficies

surface resuelve fondos transparent, subtle y accent sin logica adicional en runtime.

Transparent box Subtle box Accent box
Markup 
<nds-box surface="subtle" padding="4" radius="lg">
  <nds-text>Subtle box</nds-text>
</nds-box>

<nds-box surface="accent" padding="4" radius="lg">
  <nds-text weight="medium">Accent box</nds-text>
</nds-box>

Radios tokenizados

Puede usarse para chips, contenedores o bloques informativos sin CSS local adicional.
