Estados y contenido enriquecido

El template usa *if para el dismiss, *for para la lista y slot content para markup enriquecido authored por la app.

Review DX

Este componente sirve de smoke test real del nuevo flujo de authoring.

  • Clase corta centrada en props, estado y handlers.
  • Template legible en template.html con sintaxis parecida a Angular.
  • La lista de checklist usa *for con trackBy.
  • El dismiss re-renderiza sin listeners duplicados y emite nds-dismiss.
