nds-checkbox Events: nds-input / nds-change

Checkbox

Control binario para consentimiento, settings y formularios asociados con descripcion opcional.

Estados frecuentes

Combina checked, label, description, required y disabled.

Markup 
<nds-checkbox
  label="Recibir guia multi-stack"
  description="Incluye quickstart y recomendaciones de integracion."
  checked
></nds-checkbox>

Eventos

El host emite el estado checked para reaccionar sin acceder al control interno.

Marca o desmarca el primer checkbox para ver el log.
