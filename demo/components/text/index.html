nds-text Variants and weights

Text

Tipografia base para copy, captions, labels y texto secundario.

Variantes

La jerarquia cambia con variant y puede combinarse con weight.

Body con tag explicito para parrafos generales. Muted para informacion secundaria. Caption para metadata o labels auxiliares. Label para UI compacta.
Markup 
<nds-text tag="p">Body por defecto</nds-text>
<nds-text tag="p" variant="muted">Muted</nds-text>
<nds-text tag="small" variant="caption">Caption</nds-text>
<nds-text variant="label" weight="semibold">Label</nds-text>

Alineacion

El atributo align se refleja directamente en el host.

Left aligned text. Centered helper copy. Right aligned note.
