nds-radio-group Event: nds-change

Radio group

Seleccion unica para preferencias de stack, tipo de despliegue o decisiones excluyentes del flujo.

Opciones serializadas

El grupo genera inputs nativos manteniendo el valor seleccionado en el host y soporte para formularios.

Markup 
<nds-radio-group
  label="Prioridad de integracion"
  name="priority"
  value="astro"
  options="html:HTML base|react:React app|vue:Vue dashboard|astro:Astro marketing"
></nds-radio-group>

Eventos

nds-change expone el valor elegido desde el host del grupo.

Selecciona una opcion en el primer grupo para ver el valor emitido.
