nds-dialog Events: nds-open / nds-close / nds-cancel

Dialog

Overlay modal para confirmaciones, explicaciones de valor o pasos que necesitan foco temporal.

Controlado por el host

Abre, cierra o alterna el dialogo usando atributos y eventos del custom element.

Prueba la apertura, cierre y cancelacion sin depender de una capa extra.
Abrir dialog Cerrar dialog
Markup 
<nds-button data-dialog-demo-action="open" data-dialog-demo-target="#dialog-demo-default">
  Abrir dialog
</nds-button>

<nds-dialog id="dialog-demo-default" title="Confirmar rollout" description="Comparte el mismo paquete entre stacks.">
  <nds-text>El dialogo emite eventos publicos del host.</nds-text>
</nds-dialog>

Eventos

Escucha nds-open, nds-close y nds-cancel directamente sobre <nds-dialog>.

Abre el dialogo y pulsa Escape o el cierre para ver el log.

Dialogo dismissible

Patron para confirmaciones ligeras o explicaciones de producto.

Abre este dialogo desde cualquier pagina para explicar una decision o confirmar un paso.

Control programatico

Tambien puedes abrir y cerrar el dialogo desde botones externos para flujos guiados.

Abrir version guiada Cerrar programaticamente
