nds-field Light DOM wrapper

Field

Wrapper composable para label, descripcion, error y atributos ARIA sobre controles nativos o componentes de formulario.

Composicion de formularios

El componente conecta label, descripcion y error con el control hijo para simplificar formularios consistentes.

Markup 
<nds-field label="Email de equipo" required description="Se usa para compartir temas y tokens.">
  <nds-input type="email" placeholder="team@example.com"></nds-input>
</nds-field>

<nds-field label="Entorno objetivo" error="Selecciona un entorno antes de continuar." invalid>
  <nds-select options="prod:Produccion|staging:Staging|docs:Documentacion"></nds-select>
</nds-field>

Por que importa

nds-field deja la composicion en HTML y mantiene accesibilidad, mensajes y estado cerca del control.

  • Asocia label y descripcion con el control hijo.
  • Propaga required, invalid y error.
  • Funciona con input, textarea y select.
  • Encaja en light DOM para integrarse con formularios authored por la app.
