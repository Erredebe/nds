nds-button Event: nds-click

Button

Acciones primarias y secundarias con variantes visuales, tamanos y estado disabled.

Variantes y tamanos

La API publica combina variant, size, slot de texto y disabled.

Solid Outline Ghost Small Medium Large Disabled
Markup 
<nds-button>Solid</nds-button>
<nds-button variant="outline">Outline</nds-button>
<nds-button variant="ghost">Ghost</nds-button>
<nds-button size="lg">Large</nds-button>
<nds-button disabled variant="outline">Disabled</nds-button>

Eventos

El componente emite nds-click desde el host cuando el control interno no esta disabled.

Usa listeners sobre <nds-button>, no sobre el <button> interno.
Pulsa el primer boton para ver el log del evento.
