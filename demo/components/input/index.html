nds-input Events: nds-input / nds-change

Input

Campo controlado por atributo value con label integrado y varios tipos HTML.

Estados frecuentes

El host refleja label, placeholder, type, value y disabled.

Markup 
<nds-input
  label="Email"
  placeholder="hello@example.com"
  type="email"
></nds-input>

<nds-input label="Telefono" value="+34 600 000 000" type="tel"></nds-input>

Eventos

nds-input reacciona a cada cambio; nds-change se emite en commit del control nativo.

Escribe en el primer campo para ver el valor emitido.
